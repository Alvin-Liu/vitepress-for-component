# 使用脚手架（推荐）

# 初始化

yarn create vlib

按照提示输入项目信息后，会看到如下信息：

�  Successfully created project test-project.
�  Get started with the following commands:

$ cd test-project
$ yarn dev

# 开发

成功初始化后，会有一个简单的模板项目，并且包含一些组件开发常用的功能。

  1. yarn dev: 热更新的开发服务.
  2. yarn test: jest 单元测试.
  3. yarn build: Rollup 打包源码.
  4. yarn docs-build: 文档打包（SSR）.
  5. docs-static: 验证文档打包（通过 koa 开启静态文件服务）.
  6. yarn docs-deploy: 使用 gh-pages 以分支的形式部署到 github.io.
  7. yarn changelog: 用conventional-changelog-cli生成 changelog .
  8. yarn lint && yarn ls-lint: lint 检查.
